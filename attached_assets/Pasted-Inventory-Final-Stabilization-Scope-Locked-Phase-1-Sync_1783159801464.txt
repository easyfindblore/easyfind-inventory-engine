Issue 1 — Society Name (Highest Priority)

This is currently broken.

Society Name is frequently blank.

In one case, a Google Maps URL was written instead of the Society Name.

This must be fixed completely.

Business Rules:

If Community = Gated

→ Capture the society name whenever one is provided.

Example:

Community: Gated

Amrutha Platinum Towers

Result:

Society Name = Amrutha Platinum Towers

Never store a Google Maps URL as the Society Name.

If Community = Semi Gated

→ Capture the society name whenever one is explicitly provided.

If Standalone / Independent House

→ Leave Society Name blank.

The parser must clearly distinguish between:

* Society Name
* Google Maps URL
* Property Location

These are separate fields.
