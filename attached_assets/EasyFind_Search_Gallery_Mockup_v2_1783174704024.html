Why it looks like this: warm paper background and a deep garden-green accent instead of WhatsApp's own green or a generic real-estate red — this is meant to feel like a trusted local consultant, not a portal. The serif numerals on price are the one deliberately "premium" flourish; everything else stays quiet so the photos and facts do the work, the same restraint Airbnb and good real-estate sites use.
One technical honesty note: each card above renders as one visual block here, but on real WhatsApp (Meta Cloud API, no product catalog connected) it's actually two back-to-back messages per property — an image with a caption, then a small reply-buttons message with "Contact Now" / "More Details." There's no native swipeable carousel available without setting up a separate Meta Commerce catalog, which is out of scope for this build. Sent quickly in sequence, it reads the same to the customer as what's mocked up here.
